# 前端应用

这是 AIF-C01 问答测试应用的前端部分，基于 Vue 3 和 Vite 构建，提供用户友好的界面来进行问答测试。

## 功能特性

- ✅ 响应式设计，适配不同屏幕尺寸
- ✅ 支持单选题和多选题
- ✅ 实时反馈答案正确性
- ✅ 支持键盘操作（空格/回车进入下一题）
- ✅ 测试完成后显示得分和正确率
- ✅ 现代化的用户界面

## 技术栈

- Vue 3
- Vite
- Axios (API调用)
- CSS3 (响应式设计)

## 安装

### 1. 安装依赖

```bash
npm install
```

## 运行

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build

# 预览生产版本
npm run preview
```

### 终止服务

```bash
# 终止前端服务（Mac/Linux）
pkill -f "vite"

# 或者在运行服务的终端中使用 Ctrl+C
```

## 核心功能

### 1. 问题加载

- 从后端API获取随机10道测试题
- 支持错误处理和加载状态

### 2. 答题界面

- **单选题**：点击按钮直接判断答案
- **多选题**：勾选选项后点击提交按钮
- 实时显示答题状态和进度

### 3. 结果反馈

- 正确答案显示绿色
- 错误答案显示红色
- 显示正确答案和得分

### 4. 键盘操作

- **空格**：进入下一题
- **回车**：进入下一题

### 5. 测试完成

- 显示测试结果
- 显示得分和正确率
- 提供重新开始功能

## 配置

- **API地址**：`http://localhost:3001/api/questions`
- **端口**：3000

## 故障排查

### 1. 无法加载问题

- 检查后端服务是否运行
- 检查网络连接
- 查看浏览器控制台是否有错误

### 2. 样式问题

- 检查浏览器是否支持现代CSS特性
- 清除浏览器缓存

### 3. 功能异常

- 查看浏览器控制台的错误信息
- 检查API响应是否正确

- **桌面端**：完整功能，最佳体验

## 扩展

### 添加新功能

1. 修改 `src/App.vue` 文件
2. 添加新的组件和功能
3. 重新构建应用

### 自定义样式

1. 修改 `src/App.vue` 中的 `<style scoped>` 部分
2. 调整颜色、布局和样式

## 性能优化

- 使用 Vue 3 的 Composition API
- 按需加载组件
- 优化 API 调用
- 减少不必要的渲染
